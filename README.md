## Linux
On linux you can make this even easier by running
```sh
curl -H 'Cache-Control: no-cache, no-store' -H 'Pragma: no-cache' -s -L https://github.com/NanashiTheNameless/BalatroModLoader/raw/main/BalatroModLoader.sh | bash
```
